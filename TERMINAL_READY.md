## 🚀 **QUICK START**

### **1. Setup**
```bash
git clone https://github.com/atheendre130505/autocoder.git
cd autocoder
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.template .env
# Edit .env with your API keys
```

### **2. Run**
```bash
python3 main.py
```

### **3. Use**
```
autocoder> create Python calculator with basic math operations
autocoder> list
autocoder> help
autocoder> quit
```
